# docs/ 문서 작성 규칙

이 디렉토리(그리고 하위 디렉토리) 안의 `.md` 파일을 새로 만들거나 실질적인 내용을 추가할 때 적용된다.

## AI 작성 표기

문서 맨 아래에 `## AI 작성 표기` 섹션을 추가하고, AI(Claude)가 조사·분석·초안 작성한 부분과 사용자가 지시·검토·결정한 부분을 구분해서 짧게 적는다.

- PR 본문의 "AI 생성 부분 표기"([`docs/assignments/week-01.md`](assignments/week-01.md))와 같은 취지 — 리뷰어가 어디를 주의 깊게 봐야 하는지 알려주는 것
- AI가 코드를 읽거나 분석해서 낸 판단은 "제가 확인해보니"가 아니라 "AI가 확인해 ~라고 판단함"처럼 주체를 명확히 한다 — 사용자가 직접 알아낸 것처럼 쓰지 않는다

## 예시

```markdown
## AI 작성 표기

이 문서는 AI(Claude)가 작성했습니다. OOO 조사·분석은 AI가 수행했습니다.
어떤 방향으로 쓸지는 사용자가 지시했고, 내용은 사용자가 검토했습니다.
```
