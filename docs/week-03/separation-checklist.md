# 관심사 분류 체크리스트

> 대상: `src/productList/ProductListPage.tsx`

## 관심사 분류 표

> 코드를 훑으면서 각 코드 덩어리가 어느 관심사에 속하는지 기록한다.
> 분리 후보가 없는 줄에는 "—" 기입.

| 위치 (파일:라인) | 관심사 | 현재 레이어 | 분리 후보 | 분리하지 않을 근거 |
| ---------------- | ------ | ----------- | --------- | ------------------ |
|                  |        |             |           |                    |
