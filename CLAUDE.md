# 프로젝트 가이드

Next.js 16 + React 19 + TypeScript 기반 학습용 커머스 프론트엔드 과제 레포

## 기술 스택

- Next.js 16, React 19, TypeScript

## 명령어

- 개발 서버: `pnpm install && pnpm dev`
- 빌드: `pnpm build`
- 프로덕션 실행: `pnpm start`
- 린트: `pnpm lint`
